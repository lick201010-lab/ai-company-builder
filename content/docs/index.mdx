## 这个网站给你什么

**手把手教你在 AI 编程工具里，搭出一支多角色的 AI 团队**——CFO、CMO、COO 各司其职，单独喂资料、记得住你业务背景、互相能协作。

**适合谁：**
- 想用 AI 替代/增强一整支团队的一人创业者、自由职业者
- 听过 Claude Code / Codex / Ollama，但不知道从哪开始的小白
- 已经在用某个 AI 工具，但只跟单个 chat 对话、不会"组织化"用法的人

**不适合谁：**
- 想要"零门槛 SaaS 体验"的用户——这里没有"注册即用"的服务，需要你自己装工具、付 API 费
- 已经熟练用 subagents / multi-agent 编排的进阶用户

---

## 三条主线，选一条开始

| 路线 | 适合谁 | 月开销 | 难度 |
|---|---|---|---|
| [**Claude Code**](/docs/claude-code) | 想要最强推理 + 最成熟 agent 编排 | $20+ | ★★☆☆☆ |
| [**Codex**](/docs/codex) | 已经付 ChatGPT Plus、想沿用熟悉体验 | $20+ | ★★☆☆☆ |
| [**OpenHermes**](/docs/openhermes) | 想完全免费、本地隐私敏感、有显卡 | **0** | ★★★★☆ |

**还在犹豫？** 推荐先看 [Claude Code 路线](/docs/claude-code)——文档最全、AI 表现最稳、配套生态最完整。等跑通一次再考虑要不要换路线。
