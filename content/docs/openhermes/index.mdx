## 这条主线教什么

教你怎么用 **OpenHermes**（Nous Research 出的开源 LLM）+ **Ollama**（本地运行框架）在自己电脑上搭一支 AI 团队——**不需要任何外部 API、不烧钱、数据完全在本地**。

## 为什么选这条路

| 优势 | 说明 |
|---|---|
| **完全免费** | 没有 API 费、没有订阅费，电脑电费除外 |
| **数据隐私** | 所有对话、所有资料都不出本机 |
| **不限调用次数** | 想问多少问多少 |
| **可断网** | 不依赖任何在线服务，飞机上也能用 |

## 代价

- 需要一台**有显卡的电脑**（推荐 16GB+ 显存，最低 8GB）
- 推理质量略低于 Claude / GPT（但日常工作够用）
- 速度比云端慢，特别是上下文长的时候
- 初次配置比 Claude Code 折腾

## 适合谁

- 想完全免费试试 AI 多角色团队的人
- 处理敏感数据（客户名单、财务、合同）不想上云的人
- 有游戏本/工作站的人
- 喜欢折腾、想理解 LLM 怎么跑的人

## 不适合谁

- 没显卡 / 用笔记本核显的人——会很慢
- 想"装好就用"的小白——OpenHermes 路线折腾度比另两条主线高
- 需要最强推理能力的场景（写复杂代码、深度分析）
