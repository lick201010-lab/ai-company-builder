2. 装 Node.js（详见 [Claude Code 安装章节](/docs/claude-code/install#step-1--装-nodejs)）
3. 装 Codex CLI：`npm install -g @openai/codex-cli`（命令以官方为准）
4. `codex login` 登录
5. 在任意文件夹敲 `codex` 开始对话
