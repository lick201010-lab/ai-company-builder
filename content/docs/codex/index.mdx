## 这条主线教什么

教你用 **OpenAI Codex**（OpenAI 出的命令行 Agent 工具）搭一支 AI 团队。

跟 Claude Code 路线的核心区别：
- 用 GPT 系列模型，不用 Claude
- 体验跟 ChatGPT 网页版一脉相承
- 编排能力略弱于 Claude Code，但社区资源更广
- 跟 GPT Store、Custom GPT 互通

## 适合谁

- 已经付 ChatGPT Plus / Team / Enterprise 的人
- 习惯 OpenAI 生态、用过 Custom GPT 的人
- 国内能稳定访问 OpenAI 的人
