## 💻 IDE 集成

把 AI 团队嵌进你的代码编辑器，让"对话"变成"协作"。

<Cards>

<Card title="VS Code" href="https://code.visualstudio.com/">
免费、最主流的编辑器。Claude Code / Codex / Continue 都有 VS Code 插件。
</Card>

<Card title="Continue.dev" href="https://continue.dev/">
开源 IDE AI 插件，可以接 Claude / GPT / 本地 Ollama，多模型自由切换。
</Card>

<Card title="Cline" href="https://github.com/cline/cline">
VS Code 上的 autonomous AI agent，能直接读写文件、跑命令。
</Card>

<Card title="Cursor" href="https://cursor.com/">
专门为 AI 编程设计的编辑器（fork 自 VS Code），$20/月。
</Card>

</Cards>

## 📂 知识管理（喂资料）

把零散的资料整理成 AI 能"消化"的知识库。

<Cards>

<Card title="Notion" href="https://notion.so/">
个人知识库，可导出 Markdown 喂给 AI。免费版够用。
</Card>

<Card title="Obsidian" href="https://obsidian.md/">
本地优先的笔记工具，所有笔记都是 Markdown，AI 友好。免费。
</Card>

<Card title="AnythingLLM" href="https://anythingllm.com/">
本地 RAG 框架，把 PDF / 网页喂进去，结合 Ollama 完全离线运行。免费。
</Card>

<Card title="Open WebUI" href="https://openwebui.com/">
Ollama 的图形界面，比终端友好，支持本地 RAG。免费、开源。
</Card>

</Cards>
