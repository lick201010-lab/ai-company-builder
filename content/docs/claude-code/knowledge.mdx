## 本章学什么

- 三种喂资料方式：直接放进 CLAUDE.md / 用 skills / 用 MCP server
- 不同资料类型怎么处理：PDF、网页链接、Excel 表格、过往聊天记录
- 哪些资料该放在哪个角色名下（vs 团队共享）
- 实战：把一份"业务介绍 PDF"喂给"业务顾问"角色

## 章节结构（计划）

1. **三种喂资料的方式对比** — 10 分钟
2. **放进 CLAUDE.md（最简单）** — 10 分钟
3. **用 skills 喂大量内容** — 15 分钟
4. **用 MCP server 接外部知识源** — 提及，进阶章节深入
5. **实战：喂一份业务 PDF** — 10 分钟
