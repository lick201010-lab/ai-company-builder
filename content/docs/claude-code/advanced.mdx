## 本章学什么

- **hooks**：在 Claude 做某些动作前后，自动跑你的脚本（比如自动 git commit、自动发邮件）
- **MCP（Model Context Protocol）**：让 AI 连外部系统（飞书、Slack、Notion、数据库）
- **skills**：把一套常用操作打包成可复用的"技能"
- 实战：装一个飞书 MCP，让 AI 直接帮你发消息到群里
