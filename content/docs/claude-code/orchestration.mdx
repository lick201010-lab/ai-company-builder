## 本章学什么

- 一个角色怎么调用另一个角色（subagent 嵌套调用）
- "总裁视角"工作流：你只跟 CEO 角色对话，CEO 自动派活给各部门
- 怎么避免"AI 大锅饭"（所有角色一窝蜂回复同一件事）
- 实战：搭一个"周度复盘"工作流——CEO 角色派活，3 个 Head 各自产出后汇总
